Marine_Forecast_WebGIS
======================
Dependencies
------------------------------
- NetCDF4
- Numpy
- Shapely
- GDAL
- Matplot
- YAML
- PIL
- aggdraw

Support Marine Forecast Protucts(NetCDF)
------------------------------
- WRF
    - Wind Vector Field
    - Pressure Scalar Field
- SWAN
    - Wave Scalar Field
- WW3
    - Wave Scalar Field
- POM
    - Current Vector Field
    - Sea Level Scalar Field
- ROMS
    - Current Vector Field
    - Temperature Scalar Field
- FVCOM
    - Tide Vector Field
    - Tide Height Scalar Field

Visualisation
-----------------------------
- Scalar Field
    - Grading Image
    - Contour Line
- Vector Field
    - Grid Image
    - more to come
